## Featured Projects

### 🔵 &nbsp; [Azure Sentinel Honeypot — Live Attack Visualization](https://github.com/Pedro-Ojeda/Sentinel-Lab)
Deployed a Windows VM honeypot on Azure with intentionally exposed RDP, then used a custom PowerShell script to extract failed login events from the Windows Security Log, enrich them with geolocation data via an IP API, and forward the results into **Azure Sentinel (Microsoft Defender for Cloud)**. Built a live world map workbook to visualize inbound attack origins in real time.

`Azure Sentinel` `PowerShell` `Log Analytics` `KQL` `GeoIP Enrichment`

---

### 🔵 &nbsp; [Active Directory Home Lab — Bulk User Provisioning](https://github.com/Pedro-Ojeda/Active-Directory)
Built a fully functional Active Directory environment in VirtualBox with a Windows Server 2019 domain controller. Wrote a PowerShell script to automate bulk user creation from a name list, simulating enterprise-scale identity management and onboarding workflows.

`Active Directory` `PowerShell` `Windows Server` `VirtualBox` `Identity Management`

---

### 🐍 &nbsp; [Python CSV Log Reader](https://github.com/Pedro-Ojeda/python-csv-log-reader)
A Python utility for parsing, filtering, and summarizing exported CSV log files — useful for rapid triage when working with log exports from SIEMs or Windows Event Viewer outside of a live platform.

`Python` `Log Analysis` `CSV Parsing` `Security Automation`
